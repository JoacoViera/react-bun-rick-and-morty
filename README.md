# Rick & Morty SPA

Test project using React and [Bun](https://bun.sh/) runtime.

Bun version: 0.1.5 (beta)

## Stack

- React
- Bun

## Installing

```bash
git clone https://github.com/JoacoViera/react-bun-rick-and-morty.git
cd react-bun-rick-and-morty
bun install
```

## Run project locally

```bash
bun dev
```
